Donomo Uploader


How to run the uploader:

To run the uploader:
1. Select "Donomo Uploader", then "Command Line"
2. In the command line window, type 

uploader -u <your email address registered with donomo> -p <password> <path>

The path may be a wildcard.

Once the uploader completes when all files are submitted for processing. 
You will be notified via email when all documents are processed and are 
ready for download from https://archive.donomo.com
